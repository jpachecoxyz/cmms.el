#+title: cmms.el
#+author: Javier Pacheco
#+language: en

*cmms.el* is a simple *Computerized Maintenance Management System (CMMS)* implemented entirely inside *Emacs* using Emacs Lisp.
It provides a lightweight dashboard for managing equipment in an industrial environment directly from your editor.
This project is designed for engineers and technicians who want a minimal maintenance management workflow integrated with Emacs.

* Features
- Equipment management (add, edit, delete)
- Interactive dashboard using =tabulated-list-mode=
- Equipment status tracking
- Equipment filtering (type, area, status)
- Dynamic dashboard statistics
- Evil mode support
- Lightweight in-memory database
- No external dependencies

* Dashboard
Running the main command opens a dashboard displaying all equipment:

| ID | Name | Type | Area | Status |

The header also shows a summary of the current system:
- Total equipment
- Active equipment
- Equipment under repair
- Stopped equipment

** Example header:
#+begin_example
My Company | Equipment:12 | Active:9 | Repair:2 | Stopped:1
#+end_example

* Installation
Clone the repository:
#+begin_src bash
git clone https://github.com/jpachecoxyz/cmms.el
#+end_src

Add the directory to your =load-path=:
#+begin_src elisp
(add-to-list 'load-path "/path/to/cmms.el")
(require 'cmms)
#+end_src

* Usage
Run the main command:
#+begin_src
M-x cmms
#+end_src

This opens the CMMS dashboard.

*Keybindings*

| Key | Action                |
| --- | --------------------- |
| a   | Add equipment         |
| e   | Edit equipment        |
| d   | Delete equipment      |
| RET | Open equipment record |
| /   | Filter equipment      |
| r   | Clear filter          |
| g   | Refresh dashboard     |
| q   | Quit                  |

* Equipment Data Structure
Equipment is stored internally as an association list:
#+begin_src elisp
(EQ-001
:name "Main Motor"
:type "Motor"
:area "Production"
:status "Active")
#+end_src

* Example Workflow
1. Open the dashboard
#+begin_src
M-x cmms
#+end_src

2. Add equipment

Press:
#+begin_src
a
#+end_src

3. Filter equipment by area or type
#+begin_src
/
#+end_src
