EMMS (Emacs Maintenance Management System) is a lightweight
Computerized Maintenance Management System (CMMS) implemented
entirely in Emacs using Org Mode.

It allows users to manage:

- Areas
- Assets
- Work Orders

All data is stored in plain Org files, making it portable,
versionable, and easy to integrate with the Emacs ecosystem.

* Philosophy

EMMS follows the Emacs philosophy:

- Plain text data
- Extensible
- Hackable
- Keyboard driven
- No external database required

Everything lives inside Org files.

* Features

Current features:

- Create maintenance *Areas*
- Register *Assets*
- Generate *Work Orders*
- Asset and area validation
- Org-based data storage
- Simple dashboard interface

Planned features:

- Asset explorer UI
- Work order status tracking
- Maintenance history per asset
- Preventive maintenance scheduling
- Org Agenda integration
- Multi-user data directories
- Reporting

* Installation

Clone the repository:

#+begin_src bash
git clone https://github.com/jpachecoxyz/emms.el
#+end_src

Add it to your Emacs configuration:

#+begin_src elisp
(add-to-list 'load-path "/path/to/emms.el")
(require 'emms)
#+end_src

* Configuration

By default EMMS stores its data in:

#+begin_src
/tmp/emms/
#+end_src

You can customize this:

#+begin_src elisp
(setq emms-directory "~/emacs/emms/")
#+end_src

Files created:

| File | Purpose |
|-----|--------|
| assets.org | Areas and assets |
| workorders.org | Maintenance work orders |

* Usage

Main keybinding:

#+begin_src
C-c m
#+end_src

Available commands:

| Key | Action |
|----|-------|
| a | Create Area |
| e | Create Asset |
| w | Create Work Order |

Example:

#+begin_src
C-c m a
#+end_src

Create an area:

#+begin_src
Injection
#+end_src

Add an asset:

#+begin_src
C-c m e
#+end_src

Example:

#+begin_src
IDRA 400
Area: Injection
#+end_src

Create a work order:

#+begin_src
C-c m w
#+end_src

* Data Structure

Assets are stored inside areas:

#+begin_src org
* Injection
:PROPERTIES:
:TYPE: area
:END:

** IDRA 400
:PROPERTIES:
:TYPE: asset
:AREA: Injection
:END:

* Assembly
:PROPERTIES:
:TYPE: area
:END:
#+end_src

Work orders:

#+begin_src org
* WO 202604041530
:PROPERTIES:
:ASSET: IDRA 400
:STATUS: OPEN
:END:

Hydraulic oil leak detected
#+end_src
